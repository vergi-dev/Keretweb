# Ide másold a Verge3D fájlokat!

Ebbe a mappába várom a Verge3D export eredményét. 

### Elvárt fájlok:
- `index.html` (Fontos: ez legyen a neve!)
- `v3d.js`
- `visual_logic.js`
- `[projekt_neve].glb`
- Minden egyéb `.png`, `.jpg`, `.css` fájl, amit az export generált.

### Miután bemásoltad:
1. Nyiss egy terminált a projekt gyökérmappájában.
2. Futtasd az `npm run dev` parancsot.
3. Nyisd meg a böngészőben: `http://localhost:3000`
